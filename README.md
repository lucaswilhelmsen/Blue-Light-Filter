# Blue Light Filter

Strongest possible blue light filter for macOS.

## Note (macOS security)
Because this app is not notarized by Apple, macOS may show an “unverified developer” warning on first launch. This is normal for unsigned apps downloaded from the internet.

## Download (macOS)
[Download latest release](https://github.com/lucaswilhelmsen/Blue-Light-Filter/releases/latest)

## Installation Guide
1. Download the `BlueLightFilter-macOS.zip` file from the link above.
2. Unzip it.
3. Drag `Blue Light Filter.app` into `Applications`.
4. In `Applications`, right-click the app and choose `Open`, then click `Open` again.

If macOS blocks the app, run this in the macOS Terminal app:

```bash
xattr -dr com.apple.quarantine "/Applications/Blue Light Filter.app"
```
